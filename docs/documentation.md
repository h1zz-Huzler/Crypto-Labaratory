Документация кода CryptoLab
Общее описание

Класс CryptoLab реализует основной функционал криптографической лаборатории: шифрование/дешифрование текста, управление интерфейсом, работу с чек-листом безопасности и уведомления.
Конструктор constructor()

Назначение: Инициализация основных элементов DOM и запуск всех систем.

Что делает:

    Получает ссылки на DOM-элементы:

        inputText - поле ввода текста

        outputText - поле вывода результата

        algorithmSelect - выбор алгоритма

        keyInput - поле ввода ключа

        charCount - счётчик символов

    Запускает init() и initChecklist()

Методы инициализации
init()

Запускает все системы лаборатории:

    setupEventListeners() - настройка обработчиков событий

    setupScrollEffects() - эффекты прокрутки

    setupMobileNav() - мобильная навигация

    updateCharCount() - обновление счётчика символов

    updateAlgorithmInfo() - обновление информации об алгоритме

Настройка обработчиков событий
setupEventListeners()

Привязывает обработчики к кнопкам:

    encryptBtn - шифрование

    decryptBtn - дешифрование

    clearBtn - очистка полей

    copyBtn - копирование результата

    generateKeyBtn - генерация ключа

    inputText - обновление счётчика символов

    algorithmSelect - обновление информации при смене алгоритма

setupScrollEffects()

Управляет эффектами при прокрутке страницы:

    Добавляет класс scrolled к шапке при прокрутке > 50px

    Обновляет индикатор прогресса (scrollProgress)

    Показывает/скрывает кнопку "Наверх" (scrollTop)

    Вызывает updateActiveNavLink() для подсветки активного пункта меню

setupMobileNav()

Реализует мобильное меню:

    Открытие/закрытие меню при клике на гамбургер

    Замена иконки fa-bars на fa-times

    Закрытие меню при клике на ссылку

Навигация
updateActiveNavLink()

Подсвечивает активный пункт меню в зависимости от положения прокрутки:

    Находит все секции с id

    Определяет, какая секция сейчас в области видимости

    Добавляет класс active соответствующей ссылке

Работа с интерфейсом
updateCharCount()

Обновляет счётчик символов в поле ввода.
updateAlgorithmInfo()

Обновляет текстовое описание текущего алгоритма:

    Получает элемент #algorithmInfo p

    В зависимости от выбранного алгоритма показывает соответствующее описание

Работа с ключами
generateKey()

Генерирует случайный ключ:

    Использует символы A-Z, a-z, 0-9

    Длина ключа: 8-17 символов

    Показывает уведомление об успешной генерации

Шифрование/Дешифрование
encrypt()

Запускает процесс шифрования:

    Проверяет входные данные через validateInputs()

    Вызывает processText(true)

    Выводит результат в поле outputText

    Показывает уведомление об успехе или ошибке

decrypt()

Запускает процесс дешифрования:

    Аналогично encrypt(), но с параметром false

validateInputs()

Проверяет корректность входных данных:

    Проверяет наличие текста

    Для алгоритмов Vigenere и XOR проверяет наличие ключа

    Возвращает true/false и показывает уведомление об ошибке

processText(encrypt)

Основной метод обработки текста:

    Определяет выбранный алгоритм

    Вызывает соответствующий метод шифрования/дешифрования

    Возвращает обработанный текст

Алгоритмы шифрования
caesarCipher(text, shift, encrypt)

Шифр Цезаря:

    Поддерживает русские (32 буквы) и английские (26 букв) символы

    Для шифрования: сдвиг вперёд на shift

    Для дешифрования: сдвиг назад на shift

    Неалфавитные символы остаются без изменений

vigenereCipher(text, key, encrypt)

Шифр Виженера:

    Использует ключевое слово для определения сдвига

    Очищает ключ от неалфавитных символов

    Для каждой позиции текста использует соответствующий символ ключа

    Поддерживает русский и английский алфавиты

xorCipher(text, key)

XOR шифрование:

    Побитовая операция XOR между кодом символа и кодом символа ключа

    Ключ повторяется циклически

    Операция обратима (применяется одинаково для шифрования и дешифрования)

atbashCipher(text)

Шифр Атбаш:

    Замена первой буквы алфавита на последнюю, второй на предпоследнюю и т.д.

    А (1040) → Я (1071)

    a (1072) → я (1103)

    A (65) → Z (90)

    a (97) → z (122)

calculateShift(key)

Вычисляет сдвиг для шифра Цезаря на основе ключа:

    Суммирует коды символов ключа

    Возвращает остаток от деления на 25 + 1 (сдвиг от 1 до 25)

Вспомогательные методы
clear()

Очищает все поля:

    Очищает inputText и outputText

    Устанавливает ключ по умолчанию "секрет"

    Обновляет счётчик символов

copy()

Копирует результат в буфер обмена:

    Проверяет наличие текста

    Использует document.execCommand('copy')

    Показывает уведомление об успехе

Чек-лист безопасности
initChecklist()

Инициализирует систему чек-листа:

    Загружает сохранённое состояние из localStorage

    Настраивает обновление прогресса при изменении

    Обрабатывает кнопку сброса

    Показывает поздравление при выполнении всех пунктов

    Сохраняет состояние в localStorage при изменениях

Система уведомлений
showNotification(message, type)

Создаёт и показывает всплывающее уведомление:

    Удаляет предыдущие уведомления

    Создаёт элемент с классом notification-{type}

    Автоматически скрывается через 3 секунды

    Поддерживаемые типы: success, error, warning, info

getIcon(type)

Возвращает иконку FontAwesome для соответствующего типа уведомления:

    success: fa-check-circle

    error: fa-exclamation-circle

    warning: fa-exclamation-triangle

    info: fa-info-circle
