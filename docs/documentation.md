# Настройка интерфейса и событий
setupEventListeners()	Настраивает кнопки лаборатории	Находит все кнопки (шифровать, расшифровать, очистить, копировать, скачать, генерация ключа) и прикрепляет к ним обработчики кликов. Также отслеживает изменение выбранного алгоритма

setupNavigation()	Настраивает мобильное меню	Управляет открытием/закрытием бургер-меню на мобильных устройствах, меняет иконку с ☰ на ✕. При клике на ссылку меню автоматически закрывается

updateActiveNavLink()	Подсвечивает текущий раздел	При прокрутке страницы определяет, какой раздел сейчас виден, и добавляет класс active соответствующей ссылке в меню

setupSmoothScroll()	Плавная прокрутка к разделам	Обрабатывает клики по всем ссылкам вида #название, плавно прокручивает страницу к указанному элементу с учётом высоты шапки

setupScrollIndicator()	Индикатор прокрутки	Создаёт прогресс-бар вверху страницы, который заполняется по мере прокрутки. Также добавляет класс scrolled к шапке при прокрутке вниз

setupScrollToTop()	Кнопка "Наверх"	Показывает кнопку, когда пользователь прокрутил страницу вниз более чем на 500px. При клике плавно возвращает в начало

# Работа с текстом и примерами

setupCharacterCounter()	Счётчик символов	Считает количество введённых символов в поле ввода. Меняет цвет счётчика: жёлтый (>500), красный (>1000)

setupExamples()	Кнопки-примеры	При клике на кнопку примера заполняет поле ввода готовым текстом, выбирает нужный алгоритм и прокручивает страницу к лаборатории

setupDemo()	Демонстрационный режим	Заполняет все поля демо-данными и автоматически запускает шифрование через 0.5 секунды

# Система лайков (интерактив) Работает с багами - *

setupLikeSystem()	Инициализация лайков	Создаёт полноценную систему "лайков": проверял, ставил ли пользователь лайк ранее (localStorage), хранит общее количество лайков, обновляет счётчик *

updateLikeCount(count)	Обновление счётчика	Меняет число лайков на экране, добавляет анимацию "pop", сохраняет новое значение 

updateLikeButtonState(liked)	Состояние кнопки	Меняет внешний вид кнопки и текст ("Нравится" / "Вам нравится") в зависимости от того, поставил ли пользователь лайк

createParticles(count)	Частицы ❤️	Создаёт анимированные частицы (сердечки), которые разлетаются при клике на лайк 

showAchievement(message)	Достижение	Показывает случайное сообщение-благодарность после лайка 

# Чек-лист безопасности

setupChecklist()	Настройка чек-листа	Инициализирует чек-лист с пунктами безопасности. Загружает сохранённое состояние, настраивает обновление прогресса и кнопку сброса

updateProgress() (внутри)	Обновление прогресса	Считает количество отмеченных пунктов, сохраняет состояние. При выполнении всех пунктов показывает поздравление

# Криптографические операции

handleEncrypt()	Обработка шифрования	Берёт текст из поля ввода, выбранный алгоритм и ключ, вызывает соответствующую функцию шифрования и выводит результат

handleDecrypt()	Обработка расшифровки	Аналогично handleEncrypt(), но для расшифровки

handleClear()	Очистка полей	Очищает поля ввода, вывода и устанавливает ключ по умолчанию "секрет"

handleCopy()	Копирование результата	Копирует зашифрованный/расшифрованный текст в буфер обмена. Временно меняет иконку кнопки на ✓

handleDownload()	Скачивание результата	Создаёт текстовый файл с результатом и запускает его скачивание

generateKey()	Генерация ключа	Создаёт случайный ключ длиной 8-16 символов из букв и цифр

# Информация об алгоритмах

updateAlgorithmInfo()	Информация об алгоритме	При смене алгоритма в выпадающем списке показывает краткое описание выбранного метода шифровании

# Ядро шифрования

encrypt(text, algorithm, key)	Шифрование	Вызывает нужную функцию шифрования в зависимости от выбранного алгоритма

decrypt(text, algorithm, key)	Дешифровка	Вызывает нужную функцию дешифровки в зависимости от выбранного алгоритма

caesarCipher(text, key, encrypt)	Шифр Цезаря	Сдвигает каждую букву на величину, вычисленную из ключа. Работает с русскими и английскими буквами

vigenereCipher(text, key, encrypt)	Шифр Виженера	Использует ключевое слово для циклического сдвига букв. Поддерживает русский и английский

xorCipher(text, key)	XOR-шифрование	Применяет операцию XOR к каждому символу с соответствующим символом ключа (циклически)

atbashCipher(text)	Шифр Атбаш	Заменяет А→Я, Б→Ю, В→Э и т.д. (зеркальное отображение алфавита)

calculateShift(key)	Вычисление сдвига	Преобразует ключ-слово в число сдвига (1-25) для шифра Цезаря

# Уведомления и анимации

showNotification(message, type)	Всплывающие уведомления	Создаёт красивое уведомление (success/error/warning/info), которое автоматически исчезает через 5 секунд

setupTimelineAnimation()	Анимация истории	При прокрутке к разделу истории постепенно проявляет элементы временной шкалы

setupHistorySection()	Подготовка истории	Вызывает анимацию для временной шкалы
 
 Класс SecurityChecklist
  Отдельный класс для управления чек-листом безопасности.

constructor()	Конструктор	Находит все чекбоксы, элемент прогресса и кнопку сброса. Если чекбоксы есть, запускает инициализацию

init()	Инициализация	Загружает сохранённое состояние, добавляет обработчики событий, обновляет прогресс

loadState()	Загрузка состояния	Читает из localStorage сохранённые отметки чекбоксов и восстанавливает их

saveState()	Сохранение состояния	Записывает в localStorage текущее состояние всех чекбоксов

updateProgress()	Обновление прогресса	Считает отмеченные пункты, обновляет текст прогресса. Если все выполнены - меняет цвет и показывает достижение

showAchievement(message)	Достижение	Показывает поздравительное сообщение при выполнении всех пунктов

resetChecklist()	Сброс чек-листа	Снимает все отметки, очищает сохранённое состояние и показывает уведомление

showNotification(message, type)	Уведомление	Показывает уведомление о сбросе

addEventListeners()	Обработчики	Добавляет слушатели на чекбоксы (сохранение при изменении) и на кнопку сброса
